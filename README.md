# Friends Tracker App

A simple React project to track friends and their interaction status.

## Description

This project shows a list of friends and their details like name, status, and last contact time.  
It also includes a summary section and a simple dashboard UI.

## Technologies Used

- React.js
- Tailwind CSS
- React Router DOM
- Vite
- JSON data

## Features

- Show friends list from data
- Loading effect before data loads
- Status colors (on-track, overdue, almost due)
- Summary cards (total friends, status count)
- Simple routing system
- Responsive design
- Error (404) page

## How to Run

1. Clone the project
2. Run `npm install`
3. Run `npm run dev`
